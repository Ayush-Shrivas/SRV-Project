# SRV Media Landing Page

A responsive, professional landing page for SRV Media, built with HTML5, CSS3 (BEM), and Vanilla JavaScript.

## Setup
1. Clone or download the files.
2. Ensure the `assets/images/` folder contains the required images (WebP preferred).
3. Open `index.html` in a modern browser.

## Features
- Fully responsive design.
- Semantic HTML for SEO/accessibility.
- BEM CSS for maintainability.
- Minimal JS for mobile nav.

## Performance Notes
- Images use lazy loading.
- CSS is modular and optimized.
- No external dependencies.
